For folders (ex.: folders containing drone missions images (in subfolder or not))
* Script for extraction: timestamp_extraction.py
* CSV example: filenamesrgb_TS_lefodata.csv

For individual tif files (ex.: orthomosaics, dsm, etc.)
* Script for extraction: timestamp_extraction_file.py
* CSV example: filenamescog.csv

For individual point cloud
* Script for extraction: timestamp_extraction_laz.py
* CSV example: filenamescopc.csv
